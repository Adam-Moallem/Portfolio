# Adam Moallem Portfolio

Professional React portfolio website built with Vite.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Assets

- Place Adam's profile photo at `public/profile.png`.
- Place the CV PDF at `public/Adam-Moallem-CV.pdf`.

The hero section automatically shows a polished initials fallback until `public/profile.png` is added.
